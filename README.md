# Shadowsocks on Heroku

## 概述

用于在 Heroku 上部署 V2Ray Shadowsocks。
默认加密算法：aes-256-cfb

## 镜像

本镜像仅 6MB，比起其他用于 Heroku 的 V2Ray 镜像，不会因为大量占用资源而被封号。
